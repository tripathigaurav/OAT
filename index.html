📋 Auto-Mark Setup

To enable automatic attendance when you connect to office WiFi, pick your OS:

Manual setup steps (Mac)
  1. Download auto-attendance.sh & LaunchAgent plist
  2. Put both files in ~/.oat/ (local path — avoids cloud-sync issues)
  3. Open Terminal and run: mkdir -p ~/.oat && chmod +x ~/.oat/auto-attendance.sh && xattr -cr ~/.oat/auto-attendance.sh 2>/dev/null
  4. Fix the script path in the plist for your user: sed -i '' "s|/Users/USERNAME/PLACEHOLDER/auto-attendance.sh|$HOME/.oat/auto-attendance.sh|g" ~/.oat/com.oat.wifiattendance.plist
  5. Install the LaunchAgent: launchctl unload ~/Library/LaunchAgents/com.oat.wifiattendance.plist 2>/dev/null cp ~/.oat/com.oat.wifiattendance.plist ~/Library/LaunchAgents/ launchctl load ~/Library/LaunchAgents/com.oat.wifiattendance.plist
  6. Done! ✨ Runs automatically on WiFi change & login.
How to stop auto-tracking (Mac)

launchctl unload ~/Library/LaunchAgents/com.oat.wifiattendance.plist

Manual setup steps (Windows)
  1. Download auto-attendance.ps1 & Task XML
  2. Put both files in %LOCALAPPDATA%\OAT\ (local path — avoids OneDrive sync issues)
  3. Open PowerShell (Win+R → type powershell → Enter) and fix the path in the XML: (Get-Content "$env:LOCALAPPDATA\OAT\auto-attendance-task.xml") -replace '%LOCALAPPDATA%\\OAT', "$env:LOCALAPPDATA\OAT" | Set-Content "$env:LOCALAPPDATA\OAT\auto-attendance-task.xml"
  4. Register the scheduled task: schtasks /Create /TN "OAT-WiFiAttendance" /XML "%LOCALAPPDATA%\OAT\auto-attendance-task.xml" /F
  5. Done! ✨ Runs on network change & login.
How to stop auto-tracking (Windows)

Unregister-ScheduledTask -TaskName "OAT-WiFiAttendance"
